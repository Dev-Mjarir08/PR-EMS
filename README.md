<h2>📌 Overview</h2>

<p>
PR EMS (Employee Management System) is a <b>full-stack web application</b> built with Node.js and Express.
It helps manage <b>employees, roles, and departments</b> with a structured and scalable system.
Designed with MVC architecture, it ensures clean code and efficient data handling.
</p>

<h2>✨ Features</h2>

<ul>
  <li>👨‍💼 Employee Management (Add, Edit, Delete)</li>
  <li>🏢 Department & Role Management</li>
  <li>🔐 Authentication & Authorization (Admin/User)</li>
  <li>🖼️ Profile Image Upload</li>
  <li>⚡ Fast Rendering with EJS</li>
  <li>📂 Structured MVC Architecture</li>
  <li>🍪 JWT Authentication with Cookies</li>
</ul>
